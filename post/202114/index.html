· 2021-01-01 · # 问题实际 # 产品经理

如果给微信加一个tab, 主要的功能是今日头条的首页功能,可以吗? 为什么?

用户 -> 问题/需求/痛点(比较重要 需要满足) -> 问题模型 -> 解决方案 -> 产品

不可以.

专业技能思考

评价产品类 新功能(目标用户 + 用户需求)
1. 首先分析新功能 面对什么样的用户 解决了什么样的需求
2. 用户需求的角度
1. 需求是目标用户的痛点吗? 可不可以创造用户增长?
2. 兼容平台(产品)定位
1. 这个需求的目标用户是否可以与产品核心用户画像很好的兼容,
2. 是否兼容旧有的平台功能, 能够和谐的整合使得1+1 大于 2 而不是 1 + 1 < 2
3. 用户体验的角度

分治的思想
1. (需求拆分)想办法把问题/需求拆分 不同的问题, 用不同的方案解决,
2. (需求定级)越重要的问题, 越使用重要的笔墨精力去解决; 越小的需求 花费更少的精力解决

产品定位 == 目标用户 + 用户需求

认知提升思考

马斯诺金字塔 问题实例 -> 问题模型 在问题模型中 要解决的就是 马斯诺需求中的一层或者多层
一个问题实例 有着浅显的需求 比如社交需求 这背后是人的社交需求; 比如强票 本质是 人类对确定性的需求 对安全的需求

迭代思考 同一个问题 经过迭代越挖越深 最后找寻问题的本质 一题多做

联系思考 将不同的问题 找寻他们的相同点 多题一做
